# V3.1.2 - POS Prescription Button

This version builds on the stable V3.1.1 backend POS foundation.

## Added
- Prescription button on the POS product screen.
- Lightweight prescription capture modal in POS.
- Patient name, DOB, doctor, prescription date, pharmacist initials, notes, instructions and refill fields.
- Prescription file/photo upload from POS.
- POS order JSON export of prescription payload.
- Server-side POS order conversion to normal Pharmacy workflow.

## Important
If the POS interface does not immediately show the button after upgrade, clear browser cache/hard refresh and restart the POS session.
