# V3.1.4 POS Button Placement Patch

This patch keeps the working V3.1.3 safe POS button but moves it to a cleaner area:
- Beside Customer / Note / Upload / Options buttons
- Not inside the purple Payment bar
- Better alignment with the cashier workflow

Install by replacing the GitHub module folder, pulling/syncing in CloudPepper, upgrading the module, then hard refreshing POS with Ctrl + F5.
